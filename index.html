如何使用
gitHub地址: https://github.com/ChinaNumber-One/Yuui

说明：组件使用原生js + css 实现，开发者可根据各自业务需求修改源代码。

核心文件：yuui.js ，yuui.css

使用步骤：

1、引用所需yuui.css

<link rel="stylesheet" href="yuui.css">

2、引用所需yuui.js

<script src='yuui.js'></script>

3、调用方法 Yuui.方法() 查看代码示例

代码示例
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>代码示例</title>
        <link rel="stylesheet" href="yuui.css">
        /* btn 样式省略*/
    </head>
    <body>
        <div class="btn">显示</div>
        <div id="as"></div>
    </body>
    </html>
    <script src="https://code.jquery.com/jquery-3.1.1.min.js"></script>
    <script src='yuui.js'></script>
    <script>
        $(function () {
            let op1 = {
                domId: 'as',
                buttons: [{name:'3分',value: 3},{name:'2分',value:  2},{name:'1分',value: 1}]
            }
            Yuui.createActionSheet(op1)
                .actionSheetShow('as')
                .getActionSheetData( value => {
                    console.log(value)
            })
            $('.btn').click(function(){
                Yuui.actionSheetShow('as')
            })
        })
    </script>
ActionSheet
属性（attributes）
名字 类型 默认值 说明
domId String 空 必填 创建UI组件挂在容器的ID
title String 空 标题
desc String 空 副标题、描述等
position String bottom 菜单显示的位置。可选值：'top','middle','bottom'
buttons Array

[

{name: '1、createActionSheet(options)创建}',

{name: '2、actionSheetShow()显示}',

{name: '3、getActionSheetData(cb)获取值}',

{name: '4、actionSheetHide()关闭}'

]

菜单组列表

[{

name: '显示菜单名称',

value: '值',

name: '图标url'

}]
iconPosition String left 菜单中图标显示位置，可选值：'left'，'right'
closeOnclickMask Boolean true 点击遮罩时是否关闭菜单，适用于一些进入页面时需要强制选择的场景。
closeOnclickMenu Boolean true 点击菜单时是否自动关闭
showCancel Boolean false 是否显示“取消”菜单
concelText String 取消 “取消”菜单文字
concelTextStyle Object

{

color: '#f32836',

textAlign: 'center',

fontSize: '14px'

}

 “取消”菜单的文字样式
仅支持配置 color , textAlign , fontSize。
若有其他样式需求可自行定义样式。
buttonTextStyle Object

{

color: '#333',

textAlign: 'left',

fontSize: '14px'

}

 菜单的文字样式
仅支持配置 color , textAlign , fontSize。
若有其他样式需求可自行定义样式。
方法（methods）
名字 参数 说明
createActionSheet options - 必填 根据options参数创建Dom节点,options 以对象{"键":"值"}形式，其中domId 为必填项
actionSheetShow domId - 选填 控制actionSheet组件显示。
若传入指定domId，可控制指定actionSheet组件显示。
不传参数默认为最近一次创建的actionSheet组件
actionSheetHide domId - 选填 控制actionSheet组件隐藏。
若传入指定domId，可控制指定actionSheet组件隐藏。
不传参数默认为最近一次创建的actionSheet组件
getActionSheetData callback() - 必填 监听点击菜单并返回所选值
例：Yuui.getActionSheetData(function(value) { console.log(value) })
